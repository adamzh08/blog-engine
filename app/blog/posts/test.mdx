# Markdown Basic Syntax Test File

This document is a comprehensive test file to demonstrate **all basic Markdown syntax**. Use it as a reference or as a starting point for your own Markdown projects.

---

## 1. Headings

Markdown supports six levels of headings:

# Heading 1
## Heading 2
### Heading 3
#### Heading 4
##### Heading 5
###### Heading 6

---

## 2. Emphasis

You can add emphasis with *italics*, **bold**, or even combine them:

*Italic text using asterisks*  
_Italic text using underscores_

**Bold text using double asterisks**  
__Bold text using double underscores__

**Mixed** *emphasis* **example**

---

## 3. Blockquotes

Blockquotes are great for quoting text. You can nest them too:

> This is a blockquote.
> 
> > This is a nested blockquote.

---

## 4. Lists

### 4.1. Unordered List

- Item 1
- Item 2
  - Subitem 2a
  - Subitem 2b
- Item 3

### 4.2. Ordered List

1. First item
2. Second item
   1. Subitem 2a
   2. Subitem 2b
3. Third item

### 4.3. Task List

- [x] Completed task
- [ ] Incomplete task

---

## 5. Code

### 5.1. Inline Code

Embed code within text using backticks: `print("Hello, Markdown!")`

### 5.2. Code Blocks

Use triple backticks for longer blocks of code:

```python
def hello_markdown():
    print("Hello, Markdown!")
```

### 5.3. Code Block with Syntax Highlighting

You can specify the language for syntax highlighting:

```javascript
function greet() {
    console.log("Hello, World!");
}
```
